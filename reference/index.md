# Package index

## All functions

- [`check_packages()`](https://anthonybasooma.github.io/fwtraits/reference/check_packages.md)
  : Check for packages to install and respond to use
- [`checktrait()`](https://anthonybasooma.github.io/fwtraits/reference/checktrait.md)
  : Checks the traits spelling compared to user input.
- [`classifydata`](https://anthonybasooma.github.io/fwtraits/reference/classifydata.md)
  : Data with ecological parameters classification.
- [`clean_names()`](https://anthonybasooma.github.io/fwtraits/reference/clean_names.md)
  : Check and clean species names to match standard names in the
  database.
- [`fuzzy_codes()`](https://anthonybasooma.github.io/fwtraits/reference/fuzzy_codes.md)
  : Internal package and create dummy codes
- [`fw_be4ustart()`](https://anthonybasooma.github.io/fwtraits/reference/fw_be4ustart.md)
  : Steps to follow in using the Freshwater Information Platform
- [`fw_cite()`](https://anthonybasooma.github.io/fwtraits/reference/fw_cite.md)
  : Get database citations
- [`fw_dbguide()`](https://anthonybasooma.github.io/fwtraits/reference/fw_dbguide.md)
  : Standard table with taxonomic groups and their traits explanations.
- [`fw_fdcompute()`](https://anthonybasooma.github.io/fwtraits/reference/fw_fdcompute.md)
  : Compute the functional diversity indices and community weighted
  means.
- [`fw_fdendro()`](https://anthonybasooma.github.io/fwtraits/reference/fw_fdendro.md)
  : For computing the functional dendogram
- [`fw_fetchdata()`](https://anthonybasooma.github.io/fwtraits/reference/fw_fetchdata.md)
  : Extracting the traits from the downloaded data.
- [`fw_geosites()`](https://anthonybasooma.github.io/fwtraits/reference/fw_geosites.md)
  : Auto generation of species sites
- [`fw_keyload()`](https://anthonybasooma.github.io/fwtraits/reference/fw_keyload.md)
  : For loading the API key from the environment.
- [`fw_register()`](https://anthonybasooma.github.io/fwtraits/reference/fw_register.md)
  : Open the registration page
- [`fw_searchdata()`](https://anthonybasooma.github.io/fwtraits/reference/fw_searchdata.md)
  : To download data from the Freshwaterecology.info database.
- [`fw_setapikey()`](https://anthonybasooma.github.io/fwtraits/reference/fw_setapikey.md)
  : Function and instructions creating the API_KEY variable in the User
  environment.
- [`fw_token()`](https://anthonybasooma.github.io/fwtraits/reference/fw_token.md)
  : Access and loading the token key
- [`fw_visualize()`](https://anthonybasooma.github.io/fwtraits/reference/fw_visualize.md)
  : ggplot2 visualization.
- [`invertbackbone`](https://anthonybasooma.github.io/fwtraits/reference/invertbackbone.md)
  : Taxonomic data for species in the freshwaterecology.info database
- [`pbenthodata`](https://anthonybasooma.github.io/fwtraits/reference/pbenthodata.md)
  : Taxonomic data for the phytobentho species in
  www.freshwaterecology.info database.
- [`speciesdata`](https://anthonybasooma.github.io/fwtraits/reference/speciesdata.md)
  : Efiplus data used to develop ecological sensitivity parameters for
  riverine species in European streams and rivers.
